# MakeQuiz-TakeQuiz
**Version 0.1**
## MakeQuiz
* MakeQuiz allows you to type your own questions and answers in the textbox to create your own quiz
* MakeQuiz has no limit to the amount of questions and answers you can make
### TakeQuiz
* TakeQuiz allows you to use the quiz made in MakeQuiz
* TakeQuiz identifies the list in MakeQuiz to give you questions
* TakeQuiz takes the answerlist from make quiz to make sure you have the right answer
